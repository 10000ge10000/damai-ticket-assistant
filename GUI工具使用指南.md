# 大麦抢票GUI工具 使用指南

## 🚀 快速开始

### 启动方式（3种方式任选其一）

1. **推荐方式**：双击 `start_gui.pyw` （无CMD窗口，最干净）
2. **备用方式**：双击 `启动大麦抢票工具.bat` （有CMD窗口显示启动信息）
3. **开发方式**：命令行运行 `python damai_gui.py`

## 📋 使用步骤

### 1️⃣ 环境检测
- 启动工具后，首先点击 **"🔍 检测环境"** 按钮
- 工具会自动检查：
  - Python版本（需要3.7+）
  - selenium库是否安装
  - Chrome浏览器是否安装
- 如有问题，会弹出解决方案窗口，可一键安装缺失组件

### 2️⃣ 网页登录
- 点击 **"🔐 网页登录"** 按钮
- 工具会自动打开Chrome浏览器并访问大麦网
- 在浏览器中完成登录（扫码或账密登录）
- 登录成功后，点击弹窗中的 **"✅ 登录完成"** 按钮

### 3️⃣ 页面分析
- 在演出链接输入框中粘贴大麦网演出详情页URL
  - 格式：`https://detail.damai.cn/item.htm?id=xxxxxxx`
- 点击 **"🔍 分析页面"** 按钮
- 工具会自动分析并提取：
  - 演出名称和场地
  - 可选城市
  - 可选日期/场次
  - 价格档位
  - 售票状态

### 4️⃣ 参数配置
- 分析完成后，左侧会出现配置选项
- **选择城市**：从下拉列表选择（如果演出有多个城市）
- **选择日期**：从下拉列表选择场次
- **选择价格**：从下拉列表选择票价档位
- **自动提交订单**：谨慎勾选（建议先关闭，手动确认）
- 点击 **"✅ 确认配置"** 完成设置

### 5️⃣ 开始抢票
- 配置完成后，**"🎯 开始抢票"** 按钮会变为可用
- 点击按钮开始自动抢票
- 观察运行日志了解实时进度
- 可随时点击 **"⏹ 停止"** 中断抢票

## 📊 界面说明

### 操作步骤区域
- 显示当前进度，已完成步骤会显示绿色 ✓
- 当前进行中的步骤显示蓝色
- 出错的步骤显示红色

### 功能面板（左侧）
- **环境检测**：检查运行环境
- **演出链接**：输入目标演出URL
- **抢票配置**：选择具体参数

### 页面信息（右侧）
- 显示分析出的演出详细信息
- 包括可选项的完整列表

### 运行日志（底部）
- 实时显示工具运行状态
- 包含时间戳的详细日志
- 错误信息会标红显示

## ⚠️ 重要说明

### 购票规则
- **固定购买1张票**（无法修改数量）
- **自动选择所有观演人**（确保已在大麦APP中添加）
- 支持实名制演出的自动处理

### 安全提示
- 建议首次使用时关闭"自动提交订单"
- 手动确认订单信息无误后再提交
- 请遵守大麦网的购票规则和条款

### 环境要求
- Windows 10/11 （其他系统理论上也支持）
- Python 3.7+ 
- Chrome浏览器
- 稳定的网络连接

## 🔧 故障排除

### 常见问题

**1. 启动失败**
- 确保Python已正确安装并添加到环境变量
- 尝试用管理员权限运行
- 检查是否有安全软件阻止

**2. 环境检测失败**
- 点击自动安装按钮安装selenium
- 手动下载安装Chrome浏览器
- 重启工具重新检测

**3. 登录失败**
- 检查网络连接
- 尝试手动访问大麦网确认可正常打开
- 清除浏览器缓存重新登录

**4. 页面分析失败**
- 确认链接格式正确（详情页链接）
- 检查演出是否还在售票期内
- 尝试手动访问链接确认页面正常

**5. 抢票失败**
- 检查配置参数是否正确
- 确认演出还有余票
- 网络高峰期可能需要多次尝试

### 调试技巧
- 观察浏览器自动化过程
- 查看日志中的详细错误信息
- 可保持浏览器窗口可见以便观察

## 💡 使用技巧

### 提前准备
1. 提前在大麦APP中完成实名认证
2. 添加好观演人信息
3. 确保账户余额充足
4. 测试网络速度和稳定性

### 抢票策略
1. 提前几分钟启动工具并完成登录
2. 在开售前几分钟完成页面分析和参数配置
3. 开售时间准时点击"开始抢票"
4. 准备多个备选票价和日期

### 提高成功率
- 网络环境稳定（避免WiFi，推荐有线网络）
- 关闭不必要的网络应用
- 准备多台设备和账号（遵守购票规则）
- 选择相对冷门的场次或座位

## 📞 技术支持

如遇到问题：
1. 查看工具内的"❓ 帮助"按钮
2. 检查日志中的错误信息
3. 尝试重启工具和浏览器
4. 确认目标演出页面结构是否发生变化
