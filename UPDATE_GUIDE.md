## 📝 配置文件更新

### 新版配置示例 (`config.json`)

```json
{
  "index_url": "https://www.damai.cn/",
  "login_url": "https://passport.damai.cn/login?ru=https%3A%2F%2Fwww.damai.cn%2F",
  "target_url": "https://detail.damai.cn/item.htm?id=971053575284",
  "users": [
    "姓名1",
    "姓名2"
  ],
  "city": "长沙站",
  "dates": [
    "2025-09-28"
  ],
  "prices": [
    "288",
    "388"
  ],
  "if_listen": true,
  "if_commit_order": false
}
```

### 重要配置说明

1. **target_url**: 使用演出详情页链接（`https://detail.damai.cn/item.htm?id=xxxxx`）
2. **city**: 需要包含"站"字，如"长沙站"、"上海站"
3. **dates**: 使用YYYY-MM-DD格式
4. **prices**: 只填写数字部分，如"288"、"388"

## 🚀 使用方法

### 1. 准备工作
- 确保已安装Chrome浏览器
- 确保已安装selenium: `pip install selenium`
- 在大麦APP中预先添加好观演人信息

### 2. 配置设置
1. 修改`damai/config.json`文件
2. 填写目标演出链接、观演人姓名、城市、日期、价格等信息
3. 设置`if_commit_order`为`true`（如需自动提交）

### 3. 运行脚本
```bash
cd damai
python damai.py
```

### 4. 运行测试（可选）
```bash
python test_updated_damai.py
```

## 🛠️ 故障排除

### 常见问题

1. **登录失败**
   - 检查网络连接
   - 手动访问大麦网确认页面可正常打开
   - 确保Chrome浏览器版本较新

2. **元素定位失败**
   - 页面结构可能再次变化，需要重新分析页面源码
   - 检查目标演出是否支持在线购票

3. **购票人选择失败**
   - 确保姓名与大麦APP中保存的完全一致
   - 确保已在大麦APP中完成实名认证

4. **城市选择失败**
   - 确保城市名称包含"站"字
   - 检查演出是否在该城市有场次

### 调试技巧

1. **设置`if_commit_order`为`false`进行测试**
2. **观察浏览器自动化过程，确认每个步骤是否正确执行**
3. **查看控制台输出的错误信息**

## ⚠️ 注意事项

1. **合规使用**: 请确保遵守大麦网的使用条款，仅用于个人正常购票
2. **购票限制**: 注意演出的购票限制和实名制要求
3. **网络环境**: 确保网络稳定，避免在抢票过程中断网
4. **时间准确**: 确保系统时间准确，提前做好准备
